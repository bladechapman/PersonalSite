Toss
IEEE CapitalOne 2013 Hackathon First Place! Allows user to draw and send images from a mobile device to a computer's clipboard instantly! My work included handling network protocols, encoding, and custom drawing algorithms for the iOS app. Expect to see it out on the app store soon!
Armagetra-Snake
Powered by HTML5, Armagetra-Snake uses your mobile device as a controller for an online game of competitive snake! The game itself is rendered using HTML5 canvas and javascript, while controls are communicated through HTML5 websockets. Additionally, Google fast-buttons are used to reduce the latency of controls on mobile devices.
Air-Rise
Facebook 2013 Midwest Regional Hackathon Entry! Using similar technology to Toss, Air-rise allows users to take images through a mobile app and send it instantly to a computer's clipboard via a computer client running in the background. Computer client is cross-platform, and comes complete with fancy notifications when an image is recieved! A modified version of Base64 encoding is also used to ensure a quick transmit time.
